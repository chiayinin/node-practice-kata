# Node Practice Kata Backend API
## Description
[DEMO](https://node-practice-kata.onrender.com) <br/>
2024 年 Node.js CRUD Kata 練習。<br/>
可以使用 PostMan 打 API 如下：<br/>
- GET: `/todos`
- POST: `/todos`, `{"title": "XXX"}`
- PATCH: `/todos/{uuid}`
- DELETE: `/todos`
- DELETE: `/todos/{uuid}`
- OPTIONS
## 技術
- Node.js v20.0.9
- NPM uuid
